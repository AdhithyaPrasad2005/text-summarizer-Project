## 📌 DESCRIPTION
This module is responsible for handling user authentication in the Text Summarizer project. It allows users to securely register and log in, ensuring proper validation and password protection using hashing techniques.

---

## 📁 FILES
- **main.py** → Handles signup and login functionality  
- **models.py** → Defines the User database model  

---

## ⚙️ RESPONSIBILITIES
✔ SIGNUP API  
✔ LOGIN API  
✔ PASSWORD HASHING (bcrypt)  
✔ USERNAME VALIDATION  

---

## 🎯 PURPOSE
To provide a secure authentication system that ensures only authorized users can access the Text Summarizer application.
